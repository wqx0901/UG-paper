# 基于机器学习的预测性建模分析训练计划

## 项目简介

本项目以课题组发表的三篇论文为核心案例，结合电力能源行业的公开数据集，设计了从"基础方法"到"深度模式挖掘"再到"前沿时空建模"的进阶式训练路径。

- **目标对象**：课题组本科生（建议大二/大三，具备基础Python与机器学习知识）
- **训练周期**：12-16周（约3-4个月）
- **核心目标**：掌握时间序列预测的核心策略、深度学习在负荷/新能源预测中的应用、以及复杂时空数据的建模能力

## 训练阶段

这份详细的周计划为期 **12周**，分为三个进阶阶段。计划将**文献阅读**与**代码复现**紧密结合，旨在让本科生从“跑通代码”进阶到“理解算法逻辑”和“复现论文核心模块”。

### 工作要求

1. **每周例会 (Weekly Check-in)**：
   * **形式**：学生需准备 PPT（3-5页），展示本周的 Loss 曲线、代码片段遇到的 Bug 以及下周计划。
   * **重点**：不要只看结果，要关注“为什么模型不收敛？”“Learning rate 设大了会怎样？”等原理性问题。
2. **代码规范要求**：
   * 从 W5 开始，强制大家将数据处理、模型定义、训练循环**模块化**（写成 `.py` 文件而不是全堆在 Notebook 里）。
   * 要求添加注释，特别是输入输出张量的维度（例如 `# Input shape: [Batch, Seq_Len, Features]`）。
3. **数据集准备**：
   * 大家分工协作，做好公开数据集的清洗后版本（`.csv` 或 `.pkl`）
4. **难点预警**：
   * **W3**：PSO 优化可能会很慢，建议限制迭代次数。
   * **W7**：1D 转 2D 的维度变换。
   * **W10**：GCN 概念抽象，建议先推荐阅读 *Kipf \& Welling (ICLR 2017)* 的经典 GCN 论文作为前置知识。
